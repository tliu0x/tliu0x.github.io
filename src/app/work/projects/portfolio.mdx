## Overview

A personal portfolio website to showcase projects, share technical blog posts, and present professional experience. Built with a modern React stack and deployed with zero-config CI/CD.

## Features

- **MDX Content System** — Blog posts and project pages authored in MDX with frontmatter metadata, parsed at build time via gray-matter
- **Dynamic OG Images** — Auto-generated Open Graph images per page using next/og with custom typography
- **Dynamic Favicon** — Favicon auto-synced from GitHub profile avatar
- **Theme System** — Dark/light/system theme with persistent preference, powered by Once UI design tokens
- **Route Protection** — Optional cookie-based password protection on any route
- **RSS Feed** — Auto-generated RSS feed at /api/rss
- **SEO** — Automatic sitemap, schema.org markup, and meta tag generation

## Tech Stack

- **Next.js 16** (App Router) with React 19 and TypeScript
- **Once UI** design system for components and theming
- **MDX** for content authoring
- **Vercel** for deployment with automatic preview deployments per branch
- **Biome** for formatting, ESLint for linting
